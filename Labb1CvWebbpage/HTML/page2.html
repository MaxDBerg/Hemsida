KiwiBankomaten

Kiwibankomaten is a versatile bank app that offers a wide range of functions for both users and administrators. It allows users to manage their bank accounts and loans with ease, view account history, transfer money, and see the available amount for loaning from the bank. Users can also create new accounts and choose their preferred currency. The app's ability to convert transfers between different currencies at the current exchange rate adds to its versatility. On the other hand, administrators can manage the app more efficiently by creating new users, updating currencies, and editing account types. The program's security measures ensure that users' data is protected by limiting login attempts and blocking access after three unsuccessful tries. The program's use of local text files as a database makes it easier for administrators to save and access data between program runs. Overall, Kiwibankomaten is a user-friendly and secure bank app that provides a seamless banking experience for both users and administrators. Its versatility and aesthetic design make it an excellent choice for anyone looking to manage their bank accounts and loans conveniently.

Database Project

The Labb3Databas project is a database application that is designed to manage student and staff data for a school. The application is written in C# and uses the Entity Framework to interact with a database. The application provides a command-line interface to allow users to perform various tasks, such as adding new staff members or retrieving a list of students in a particular class. The application also allows users to sort the list of students based on either their first name or last name in ascending or descending order. The project provides a starting point for a school administration system that can be further developed and customized to meet specific school needs.

Simple Console Bank App

This is a simplified bank app that has fewer features than the previous one. It has a simple menu with four options: view account balance, transfer funds, withdraw money, and log out. To use the app, a user needs to log in by entering their username and password. The app then gives the user three chances to log in before terminating. Once the user logs in, they are presented with the main menu. Option 1 on the menu allows the user to view their account balance. This option prompts the user to enter their username and password and then displays their account balance. Option 2 allows the user to transfer funds between their accounts. This option prompts the user to enter their username and password, then displays a list of their accounts with their current balances. The user can select the account to transfer funds from and the account to transfer funds to, then enter the amount to transfer. Option 3 allows the user to withdraw money from their account. This option prompts the user to enter their username and password and then displays a list of their accounts with their current balances. The user can select the account to withdraw money from and enter the amount to withdraw. Option 4 allows the user to log out of the app. If the user inputs an invalid menu option, the app displays a message prompting the user to input a valid option. The app also forces the user to press Enter to return to the main menu after completing an action.
